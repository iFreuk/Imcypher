⚠️ How to Share Safely

Messaging apps like WhatsApp, Messenger, and Telegram compress images, which destroys hidden messages.

To keep the message:

You must send the image as a FILE / DOCUMENT.

  • Android: Tap Paperclip ➔ Select Document ➔ Pick the image.
  • iOS: Save to Files ➔ Tap + ➔ Select Document.
  • Desktop: Drag and drop as a file, or select "Document" option.

If you send it as a normal "Photo", the message will be lost!
